# 🎓 EduCore LMS

EduCore is a modern Learning Management System (LMS) built with FastAPI and PostgreSQL.

## ✨ Features

### 👨‍🏫 Teacher Panel

* Authentication with JWT
* Face ID login
* Student management
* Attendance management
* Group management
* Payment management
* Coin system
* Statistics

### 👨‍🎓 Student Panel

* Login using Student ID and password
* Attendance history
* Payment history
* Coin balance
* Group information

### 👨‍💼 Admin Panel

* SQLAdmin dashboard
* User management
* Group management
* Teacher management
* Student management
* Attendance and payment control

## 🔐 Authentication

* Access Token
* Refresh Token
* Face ID Login
* Role Based Permissions

## 🛠 Technologies

### Backend

* FastAPI
* SQLAlchemy
* Alembic
* PostgreSQL
* SQLAdmin
* InsightFace
* JWT Authentication

### Frontend

* React + TypeScript

## 🚀 Deployment

* Backend: Render
* Database: Supabase PostgreSQL
